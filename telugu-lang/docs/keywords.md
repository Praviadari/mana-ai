# Telugu Language Keywords (v0.1)

Complete list of reserved keywords in `telugu-lang` and their JavaScript equivalents.

| Telugu keyword   | Pronunciation      | JavaScript equivalent | Description                        |
|------------------|--------------------|-----------------------|------------------------------------|
| `చెప్పు`         | cheppu             | `console.log`         | Print a value to the console       |
| `అయితే`          | ayithe             | `if`                  | Conditional branch                 |
| `లేదంటే`         | ledanṭe            | `else`                | Alternative branch                 |
| `అయ్యే_వరకు`     | ayyē_varaku        | `while`               | Loop while condition is true       |
| `పని`            | pani               | `function`            | Declare a function                 |
| `తిరిగి`         | tirigi             | `return`              | Return a value from a function     |
| `వేరియబుల్`      | vēriẏabul          | `let`                 | Declare a mutable variable         |
| `స్థిరం`         | sthiraṁ            | `const`               | Declare a constant (immutable)     |
| `నిజం`           | nijaṁ              | `true`                | Boolean true                       |
| `అబద్ధం`         | abaddhaṁ           | `false`               | Boolean false                      |
| `మరియు`          | mariyu             | `&&`                  | Logical AND                        |
| `లేదా`           | lēdā               | `\|\|`                | Logical OR                         |
| `కాదు`           | kādu               | `!`                   | Logical NOT                        |

---

## Operator symbols

The following standard ASCII operator symbols are also recognised:

| Symbol | Meaning          |
|--------|------------------|
| `=`    | Assignment       |
| `==`   | Equal            |
| `!=`   | Not equal        |
| `<`    | Less than        |
| `>`    | Greater than     |
| `<=`   | Less or equal    |
| `>=`   | Greater or equal |
| `+`    | Addition         |
| `-`    | Subtraction      |
| `*`    | Multiplication   |
| `/`    | Division         |
| `%`    | Modulo           |

---

## Reserved word status

All Telugu keywords in the table above are **reserved** — they cannot be used as variable or function names.
ASCII identifiers are not reserved and may be freely used alongside Telugu-script identifiers.
