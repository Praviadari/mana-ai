# Model License Guidance

This repository may include model-related assets in the future (checkpoints, adapters, prompts, eval artifacts).

## Default Policy

- Code in this repository is licensed under Apache-2.0.
- Model weights/checkpoints may carry **separate licenses**.
- Dataset-specific terms always take precedence for data usage.

## Recommended for Future Releases

For each model release, include:

1. `MODEL_CARD.md`
   - Intended use
   - Out-of-scope use
   - Training data summary
   - Known limitations/biases
   - Evaluation summary

2. `MODEL_LICENSE.txt`
   - Exact model license terms
   - Commercial usage statement
   - Redistribution conditions

3. Safety notes
   - Misuse risks
   - Mitigations
   - Reporting process

## Interim Rule

Until model licenses are explicitly added for a release, assume **no permission is granted** to use unpublished model artifacts.
