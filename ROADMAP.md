# Roadmap

## Phase 1 (0-3 months): Telugu Language MVP

- Define Telugu keyword set and grammar
- Implement lexer, parser, and AST
- Build JavaScript transpiler
- Create CLI (`mana run file.tel`)
- Publish 50+ learning examples in Telugu

## Phase 2 (3-6 months): Developer Tooling

- VS Code syntax highlighting
- Linter and formatter
- Better error diagnostics (Telugu + English)
- Browser playground demo

## Phase 3 (6-12 months): Telugu Coding Intelligence

- Build Telugu code-instruction dataset
- RAG-based coding assistant for language docs
- Fine-tune small coding models for Telugu prompts
- Benchmark coding quality and correctness

## Phase 4 (12-24 months): Telugu LLM Expansion

- Broaden corpus beyond coding (education, civic, STEM)
- Instruction tuning and alignment
- Evaluate reasoning, factuality, and safety in Telugu
- Release open checkpoints, evals, and reproducible pipelines

## Milestones

- M1: Grammar + parser stable
- M2: End-to-end transpile and run
- M3: Tooling alpha release
- M4: Telugu coding assistant beta
- M5: Telugu-focused LLM baseline release
