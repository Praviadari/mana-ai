# mana-ai

**mana-ai** is an open-source initiative to build:

1. **A Telugu-based programming language** (`telugu-lang/`)
2. **A Telugu-focused LLM research stack** (`telugu-llm-lab/`)

The long-term goal is to enable high-quality Telugu-first developer tooling, education, and AI systems.

## Repository Structure

- `telugu-lang/` — language design, compiler/transpiler, CLI, examples
- `telugu-llm-lab/` — datasets, training recipes, evaluation, model experiments
- `docs/` — shared design docs and governance

## Getting Started (Planned)

### Telugu Language Track
- Define keywords and grammar
- Build lexer and parser
- Transpile Telugu syntax to JavaScript
- Add CLI + diagnostics in Telugu

### Telugu LLM Track
- Curate Telugu datasets with clear licensing
- Build tokenizer and evaluation benchmarks
- Fine-tune open models for Telugu tasks
- Publish reproducible training and eval pipelines

## Vision

Build a sustainable Telugu AI ecosystem:
- Telugu programming for beginners and students
- Telugu coding assistant capabilities
- Telugu-first foundation model research
