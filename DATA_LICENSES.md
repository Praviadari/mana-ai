# Data Licenses and Provenance

This file tracks licensing and provenance for all datasets used in `mana-ai`.

## Core Principles

1. **No unclear-license data**
2. **No private or sensitive personal data**
3. **Document provenance for every dataset**
4. **Respect original attribution and share-alike terms**

## Dataset Entry Template

For each dataset, add an entry with:

- Name:
- Source URL:
- Maintainer/Owner:
- License:
- Allowed Uses:
- Restrictions:
- Contains Personal Data (Yes/No):
- Notes:

## Usage Rules

- Only include datasets with explicit, compatible licenses.
- If license is ambiguous, do not include until clarified.
- Keep raw and processed data lineage documented.
- Remove data immediately if valid takedown/legal issue is reported.
